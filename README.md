# mytravle_vue_demo

> A Vue.js project
> 第一次使用vue.js做一个小的测试项目

## 安装步骤

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
